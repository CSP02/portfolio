Toggler:

Toggler is another project which can alternatively change the classes, image sources and properties of an element. This package can also toggles the sliding of an element(like toggleSlide() method in jquery). Used vanillaJS to create to package.

Methods:

Methods is another js package which saves the time of user while they are writing same code so many times. Basically HTML has a tag called object which can be used to use a block of code in a file and the block should be on same file. But methods can import the blocks from another files as well as same file with just two lines of code and some attributes in HTML.

x86OC

x86OC is a instruction to opcode (hexcode) converter. This package can convert the x86 microprocessor instructions to their respective opcodes and send the hexadecimal codes of the instruction. Used nodeJS to create x86OC.
